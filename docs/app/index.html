Go live with your own key

Connect an OpenRouter key to run a real model. Your key is stored only in this browser and is sent only to OpenRouter — never to us, never to any server of ours.

openai/gpt-4o-mini and deepseek/deepseek-chat are cheap and work on a tiny balance; anthropic/claude-3.5-sonnet is stronger but pricier. Browse the full catalogue at openrouter.ai/models.
Low on credits? Add a few dollars at openrouter.ai/settings/credits — or just use demo mode, which needs no key. Lumen caps output per turn so small balances still work.
No key? Just close this — demo mode works with zero setup.
